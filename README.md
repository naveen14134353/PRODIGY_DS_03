## Task 3: Decision Tree Classifier

In this task, I built a Decision Tree model to predict whether a customer will take a loan based on their demographic and financial data.

## 1. Objective

To build a classification model that predicts customer decision behavior (loan: Yes/No) using a Decision Tree.

## 2. Data Preprocessing

* Handled missing values using mean imputation
* Converted categorical variables using Label Encoding
* Selected relevant features for model training

## 3. Model Building

* Split dataset into training and testing sets
* Trained a Decision Tree Classifier using sklearn

## 4. Model Evaluation

* Tested the model on unseen data
* Evaluated accuracy of predictions

## 5. Tools and Technologies

* Python
* Pandas
* Scikit-learn

## 6. Key Learnings

* Understanding Decision Tree working
* Importance of data preprocessing
* Handling categorical data
* Train-test splitting and evaluation
